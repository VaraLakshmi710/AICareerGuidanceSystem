# AI Career Guidance System

## Project Description
The AI Career Guidance System helps students and job seekers identify suitable career paths based on their skills, interests, and educational background.

## Features
- Career recommendations based on user input
- Skill assessment and analysis
- AI-powered guidance
- User-friendly interface
- Personalized career suggestions

## Technologies Used
- Java
- HTML
- CSS
- JavaScript

## Project Structure
- src/ - Source code
- README.md - Project documentation

## How to Run
1. Download or clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run the main Java file.
4. Follow the on-screen instructions.
